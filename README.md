# Bookstack export spliter
split your Bookstack HTML-books like this
- Book > (Chapter) > Page
*(a chapter is not necessary)*

## Requirements
* Python
* Bookstack (tested on v0.26.2 )
* HTML-Exported book(s)

## Usage
```sh
$ git clone <repo-url>
cd <reponame>
$ cp <html books> ./html
$ ./split.py
```
